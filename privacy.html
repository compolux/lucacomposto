Privacy Policy

This website is committed to respecting your privacy.

1. No Personal Data Collected

I do not collect, process, or store any personal data from visitors. You can browse this site freely without providing any information.

2. No Cookies or Tracking

This website does not use cookies, analytics tools, or third-party trackers. Your visit is not monitored or analyzed.

3. Locally Hosted Fonts

All fonts used on this site are hosted locally. No data is shared with external font providers such as Google Fonts.

4. Server and Hosting

The website is self-hosted on a private server provided by Contabo GmbH. The server runs Docker containers with Nginx Webserver and Reverse Proxy based on the latest Alpine Linux.

5. DNS and Proxy Services

I have chosen Cloudflare as a reverse proxy and DNS provider. Only basic DNS services are configured. Cloudflare may process technical connection data (such as IP addresses) as part of standard DNS and proxy functionality. Please refer to https://www.cloudflare.com/privacypolicy/ for more information.

6. Search Engine Visibility

This website is registered with Google Search Console and Microsoft Bing webmaster tools to improve visibility on search engines. A `robots.txt` file is provided to guide crawlers. No personal data is transmitted or stored in this process.

7. External Links

If this website contains links to external websites, I'm not responsible for their content or their privacy practices.

8. Contact

If you have any questions regarding this privacy policy, you can contact me directly via the contact details provided on this website.

9. GDPR Compliance

This website is operated in accordance with the principles of the General Data Protection Regulation (GDPR - Regulation (EU) 2016/679). Since we do not collect or process any personal data, no data processing agreement (DPA) is required, and no consent banner is necessary. If this changes in the future, the privacy policy will be updated accordingly.

This privacy policy may be updated occasionally. Please check back for changes.
